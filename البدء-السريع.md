## 2️⃣ 5 خطوات للبدء

### ✅ الخطوة 1: ملء الاسم
```
⚡ اكتب اسمك في حقل "الاسم (عربي)"
   سيظهر فوراً على البطاقة
```

### ✅ الخطوة 2: رفع الصورة
```
⚡ اضغط على صندوق الصورة
   اختر صورة من جهازك
   ستظهر معاينة فوراً
```

### ✅ الخطوة 3: إضافة البيانات
```
⚡ أكمل ملء:
   - رقم الهوية
   - التواريخ
   - البيانات الإضافية
```

### ✅ الخطوة 4: التعديل (اختياري)
```
⚡ اضغط على أي حقل على البطاقة
   عدّل الموضع والحجم من اليمين
```

### ✅ الخطوة 5: التحميل
```
⚡ اضغط زر "تحميل البطاقة"
   ستُحفظ كصورة PNG
```

## 🎯 الاستخدام الأساسي

### المعاينة الفورية
```
كل تغيير تكتبه يظهر على الفور
لا حاجة للضغط على "معاينة" مثل القديم
```

### تحريك الحقول
```
في v2: استخدم الأسهم من لوحة التحكم
في Advanced: اسحب الحقل مباشرة بالماوس
```

### تغيير الألوان والأحجام
```
اختر الحقل → اختر اللون من لوحة الألوان
أو استخدم منزلقات الحجم والخصائص
```

## ❓ أسئلة شائعة

### س: كيف أرفع صورة؟
```
ج: اضغط على صندوق الصورة بالكامل
   اختر ملف من جهازك
   أو اسحب الصورة مباشرة (في Advanced)
```

### س: كيف أحفظ التصميم؟
```
ج: البيانات تُحفظ تلقائياً في المتصفح
   إذا أغلقت التطبيق ورجعت سيكون كل شيء محفوظ
```

### س: كيف أحذف حقل؟
```
ج: في Advanced: اختره واضغط "حذف"
   في v2: قم بتعديل الحقول مباشرة
```

### س: هل أستطيع إضافة حقول جديدة؟
```
ج: في الإصدار الحالي لا (قريباً)
   لكن يمكن تعديل الحقول الموجودة
```

### س: كيف أُصدّر بجودة عالية؟
```
ج: اضغط "تحميل البطاقة"
   يُصدّر بـ 2x الدقة تلقائياً
```

### س: هل يعمل على الجوال؟
```
ج: نعم! الإصدار v2 مثالي للجوال
   في Advanced بعض الميزات أفضل على الحاسوب
```

### س: ماذا إذا واجهت خطأ؟
```
ج: 1. أعد تحميل الصفحة (Ctrl+Shift+R)
   2. امسح بيانات المتصفح
   3. استخدم متصفح مختلف
   4. جرب النسخة الأخرى
```

### س: هل البيانات آمنة؟
```
ج: نعم! لا توجد بيانات ترسل للإنترنت
   كل شيء محفوظ محلياً في جهازك
```

### س: ما الفرق بين الإصدارين؟
```
ج: v2: بسيط وسريع للمبتدئين ✅
   Advanced: متقدم مع Drag & Drop الكامل ✅
   
   ابدأ بـ v2 ثم جرب Advanced لاحقاً
```

## 🔧 إعدادات متقدمة

### في المتصفح:

**لحذف البيانات المحفوظة:**
```
Chrome: ⚙️ Settings → Privacy → Clear browsing data
Firefox: ☰ Menu → History → Clear Recent History
Safari: ⚙️ Settings → Privacy → Manage Website Data
```

**لتفعيل الشبكة الدليلية (Advanced فقط):**
```
اضغط Ctrl+G لإظهار الشبكة
مفيدة جداً للتوضع الدقيق
```
